PROJECT TITLE: Knight's Tour
PURPOSE OF PROJECT:
To move across a chess board as a knight trying to reach every square one time
VERSION or DATE: 1.0
HOW TO START THIS PROJECT: Open the Bluj Package, Run the game class
AUTHORS: Daniel Malis
USER INSTRUCTIONS:
1.) Open the Bluej Package
2.) Run the game class
3.) Enter in the number for the spot on the board you want to move to
4.) k represents the knight and x is for squares already reached
