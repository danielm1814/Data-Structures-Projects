World's Hardest Game
By Daniel Malis
Version 1.0

Instructions to open:

1. Unzip the folder that contains the __init__ (Everything is in that one class)
2. Go to the command prompt if you are on windows and pip install arcade
3. Open up the program using an IDE to see the code -- It was built in pycharm
4. Run the __init__ and the instructions and game will start loading
